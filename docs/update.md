# ロボットのソフトをアップデートする

次のコマンドでロボットのソフトをアップデートできます。

```shell
roscd oit_minibot_middle_02/
git pull
```

`git pull`を何回か実行して、`Already up to date.`という表示が出たら最新の状態になっています。

同じように以下のコマンドも実行してください。

```shell
roscd oit_roboclaw_driver/
git pull
```
