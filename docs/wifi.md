# 無線LANへの接続とオンラインマニュアルの閲覧

デスクトップ左下のボタンをクリックし、「設定」から「高度なネットワーク設定」を実行してください。

![2025-03-08_200053.png](../images/2025-03-08_200053.png)

ウィンドウ左下の「＋」マークを押してください。

![2025-03-08_200401.png](../images/2025-03-08_200401.png)

接続の種類を「`Wi-Fi`」にして「作成」をクリックします。
